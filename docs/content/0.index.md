#title
PHP Cloudflare API

#description
A simple Object Oriented wrapper for [Cloudflare API](https://developers.cloudflare.com/api), written with PHP.

#extra
  ::list
  - PHP >= 8.0
  - Minimal API around the [Guzzle HTTP client](https://github.com/guzzle/guzzle)
  ::

  ---
